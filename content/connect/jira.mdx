# Connect Jira

CodeNow connects to Jira via the Atlassian MCP server. Tickets become first-class context for AI prompts.

## Quick connect

1. Open the **Connections** sidebar tab
2. Click **Jira** → Atlassian OAuth
3. Authorize the projects you want CodeNow to access

## What the AI can do

- Pull ticket details, comments, descriptions into prompts
- Search across projects for related issues
- Generate Jira-formatted output (markdown for descriptions, ADF for comments)

## What's coming

- Bi-directional sync — ticket status updates from agent runs
- Sprint-aware planning agents
- Confluence integration alongside Jira
