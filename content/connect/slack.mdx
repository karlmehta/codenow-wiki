# Connect Slack

CodeNow connects to Slack via the official Slack MCP server, scoped per-project.

## Quick connect

1. Open the **Connections** sidebar tab in CodeNow
2. Click **Slack** → OAuth handoff to your workspace
3. Authorize the channels CodeNow can read
4. Done — the AI in this project now sees Slack context

## What the AI can do once Slack is connected

- Read messages from authorized channels
- Send messages on your behalf (with explicit prompts)
- Search across channels for context
